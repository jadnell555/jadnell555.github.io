GitHub: GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.
~ GitHub.com
Jupyter Notebooks: Jupyter Notebook is a simplified notebook authoring application for interactive computing with computational notebooks. ~ Jupyter-Notebook.readthedocs.io
